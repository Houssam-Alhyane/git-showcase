# Git Learning Project

## Overview

This project is designed to introduce you to the world of version control and collaboration using Git. Git is a powerful and widely used tool for tracking changes in your projects, collaborating with others, and ensuring the integrity of your code.

Throughout this project, you will embark on a journey of progressively building your Git skills. Starting from the basics, you'll gradually explore more advanced topics, equipping yourself with the essential knowledge and practices for effective version control and collaboration.

**Let's Git ready for it!**

---

## Project Structure

This project covers the following Git concepts and operations:

1. **Setting Up Git** - Installation and initial configuration
2. **Git Commits** - Creating and managing commits
3. **History** - Viewing and formatting commit history
4. **Check It Out** - Navigating between commits
5. **Tagging** - Creating and managing version tags
6. **Undoing Changes** - Reverting, resetting, and amending commits
7. **Moving Files** - Reorganizing project structure with Git
8. **Git Internals** - Understanding blobs, trees, and commits
9. **Branching** - Creating and managing branches
10. **Conflicts & Merging** - Resolving conflicts and merging strategies
11. **Remote Repositories** - Working with local and remote repositories
12. **Bare Repositories** - Understanding and using bare repositories for collaboration

---

## Documentation

Each exercise is documented with the commands used and the process followed. The documentation includes:

- Commands executed for each task
- Challenges faced and solutions implemented
- Lessons learned throughout the project

---

## Evaluation Criteria

The project is evaluated based on:

- **Correctness** of the Git commands used
- **Clear understanding** of Git commands and concepts
- **Commit history** reflecting the changes made throughout the exercises
- **Documentation** detailing the process followed
